# Fokus ⏱️

Aplicação de produtividade baseada na técnica Pomodoro, desenvolvida com HTML, CSS e JavaScript puro.

## 🔗 Acesse o projeto

[projeto-fokus-omega-five.vercel.app](https://projeto-fokus-omega-five.vercel.app)

## 📋 Funcionalidades

- Temporizador com três modos: **Foco** (25 min), **Descanso curto** (5 min) e **Descanso longo** (15 min)
- Música de fundo opcional durante o timer
- Sons de play, pause e notificação ao finalizar
- Troca de contexto com alteração de tema visual e imagem
- Lista de tarefas com **CRUD completo**:
  - Adicionar, editar e remover tarefas
  - Marcar tarefa como concluída automaticamente ao finalizar o foco
  - Limpar tarefas concluídas ou todas as tarefas
- Tarefa em andamento exibida em destaque
- Persistência de dados com **localStorage**
- Design responsivo para desktop, tablet e mobile

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
